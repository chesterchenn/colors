# colors
This is a express + react app.

## 目录结构
```
. 
|--API               # API目录
|--config            # 颜色配置
|--UI                # UI展示
|  |-- public      
|  |-- src
|--Manage            # 后台管理
```

### 启动API
进入api目录 `cd api`  
安装依赖 `npm install`  
运行api项目 `npm run start`  

### 启动UI
进入UI目录 `cd ui`  
安装依赖 `npm install`  
运行UI项目 `npm run start`
