# colors
This is a express + react app.

## 目录结构
```
. 
|--API                # API目录
|--UI                 # UI目录
   |-- mock           # MOCK数据
   |-- public         # 公共文件
   |-- src 
       |-- compoents  # 组件
       |-- config     # 配置文件
       |-- containers # 容器
       |-- routes     # 路由页面
       |-- redux      # REDUX文件
```

### 启动API
进入api目录 `cd api`  
安装依赖 `npm install`  
运行api项目 `npm run start`  

### 启动UI
进入UI目录 `cd ui`  
安装依赖 `npm install`  
运行UI项目 `npm run start`
